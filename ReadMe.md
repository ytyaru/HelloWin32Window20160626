# このソフトウェアについて #

HelloWin32Window20160626は私個人が学習目的で作成したソフトウェアです。

WindowsAPIを使ってWindowを表示しました。
以後、これを使いまわしてWindowプログラミングできそう。

# 開発環境 #

* Windows XP SP3
* C++/Win32
* Visual C++ 2010 Express
